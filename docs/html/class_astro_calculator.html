void AstroCalculator::UpdateLst ( double jd)

Update the local sidereal time.

Update the local sidereal time based on the given Julian date. The Julian date is the number of days since the beginning of the Julian period, which started on January 1, 4713 BC. The Julian date is used in astronomy to avoid the complexities of the Gregorian calendar.

Parameters
jdThe Julian date.
