Detailed Description

A class that controls the movement of the pointer.

The class controls the movement of the pointer by controlling the azimuth and altitude motors. The angles are specified in radians and are measured with respect to the device's, as opposed to the observer's, north and zenith. The azimuth angle is measured eastward from the north, and the altitude angle is measured up from the horizon.

The class is a singleton class.
