Detailed Description

A class that provides access to a database of stars.

The class provides functions to search for stars by name, right ascension, and declination. It uses a ball tree data structure to efficiently search for stars within a certain radius, and it uses a trie data structure to search for stars by name.

The database is stored on the flash memory of the device as binary files:

  • db.bin: Contains the star database entries.
  • ball_tree.bin: Contains the ball tree nodes.

The class is a singleton class.
