\doxysubsection{Detailed Description}
A class that calculates the positions of celestial objects. 

The class calculates the positions of celestial objects based on the observer\textquotesingle{}s location and the current time. The observer\textquotesingle{}s location is specified by the latitude and longitude, both in radians. The current time is specified by the Julian date, which is the number of days since the beginning of the Julian period, which started on January 1, 4713 BC.

The class is a singleton class.

Gets the horizontal components of the normalized earth\textquotesingle{}s magnetic field in the Earth\textquotesingle{}s reference frame. 

This function calculates the horizontal components of the Earth\textquotesingle{}s magnetic field based on the observer\textquotesingle{}s location and the current time, using the World Magnetic Model (WMM).

The Earth\textquotesingle{}s reference frame has the X-\/axis pointing east, the Y-\/axis pointing north, and the Z-\/axis pointing up.
