\doxysubsection{Detailed Description}
A class that provides access to various sensors. 

The \doxylink{class_sensors}{Sensors} class provides access to various sensors, such as the accelerometer, magnetometer, and GPS. Think of this class as an abstraction layer that hides the details of the sensors from the rest of the program. The class provides methods to read data from the sensors and to calibrate the sensors.

The implementation assumes that the accelerometer is an ADXL345 and the magnetometer is a QMC5883. Both of those sensors are connected to the I2C bus. The GPS module is assumed to be connected to UART0 but can be customized in \doxylink{class_sensors_a66a7e65d6e041de576028f97b8379594}{Init()}.
